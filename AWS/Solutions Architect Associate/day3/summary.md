## Core Services at a Glance
- EC2: Virtual servers. Use ASG + ELB for HA/scale.
- S3: Durable object storage. Use lifecycle & versioning.
- RDS: Managed relational DB. Use Multi-AZ for HA.
- VPC: Network isolation. Use private subnets for DBs.
- Lambda: Serverless compute for event-driven systems.
- IAM: Use roles and least privilege.

## Quick Tips
- Prefer managed services to reduce operational burden.
- Use CloudWatch + X-Ray for observability on distributed systems.
- Encrypt sensitive data at rest using KMS.
- Monitor costs with Budgets and Trusted Advisor.
